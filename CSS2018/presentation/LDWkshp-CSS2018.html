Which to choose: Neo4j or RDF?

Neo4j

  • Graph path traversal
  • Process flow
  • Changing (transactional) data
  • Where connections/relations(links) are key

Which to choose: Neo4j or RDF?

RDF

  • Classification (ontologies)
  • Rules and Logic
  • Datatyping, Time concepts
  • Non-transactional data
